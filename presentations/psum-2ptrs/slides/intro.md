# O que veremos hoje

Técnicas clássicas para consultas e buscas eficientes em arrays:

**Prefix Sum (Soma de Prefixos)**
- Responder somas de intervalos em $O(1)$

**Prefix XOR**
- XOR de intervalos em $O(1)$

**Two Pointers**
- Dois ponteiros que percorrem o array juntos: $O(N)$

**Sliding Window (Janela Deslizante)**
- Subarray de tamanho fixo ou variável: $O(N)$

**Prefix Sum 2D**
- Somas de subrregiões de matrizes em $O(1)$

::right::

**Pré-requisitos:** arrays, loops, complexidade básica

**Por que importa?**

Esses padrões aparecem em **centenas** de problemas de CP e são base para técnicas mais avançadas (BIT, Segment Tree...).

| Técnica | Pré-processamento | Query |
|---|---|---|
| Prefix Sum | $O(N)$ | $O(1)$ |
| Prefix XOR | $O(N)$ | $O(1)$ |
| Two Pointers | — | $O(N)$ |
| Sliding Window | — | $O(N)$ |
| Prefix Sum 2D | $O(NM)$ | $O(1)$ |

# Seg Tag

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 2rem; margin-top: 2rem;">
<div>

A Tag é um template que guarda as informações de update para cada nó da segtree.

O Tag tem 2 responsabilidades:

<li> Guardar o update pendente </li>
<li> Saber se unir com outra tag </li>

O node da SegLazy precisa saber aplicar a tag em range.
</div>

<div style="--slidev-code-font-size: 0.5em; transform: translateY(20px);">

````md magic-move
```cpp {*}{maxHeight:'400px'}
struct Node { // SegTree
    ll val = 0;
    Node(ll v = 0) : val(v) {}

    static inline Node merge(const Node& l, const Node& r) {
        return Node(l.val + r.val);
    }

    inline void apply(int v) {
        val += v;
    }
};
```
```cpp {all|1-12|14-20|all}
struct Node { // SegLazy
    ll val = 0;
    Node(ll v = 0) : val(v) {}

    static inline Node merge(const Node& l, const Node& r) {
        return Node(l.val + r.val);
    }

    inline void apply(const Tag& t, int l, int r) {
        val = val + (r - l + 1) * t.add;
    }
};

struct Tag {
    ll add = 0;

    inline void compose(const Tag& t) {
        add += t.add;
    }
};
````

# O que é uma SegTree?

A Segment Tree (SegTree) é uma estrutura de dados que permite realizar consultas de intervalo e atualizações de elementos de forma eficiente.

- **Estrutura:** Árvore binária quase completa.
- **Fundamento:** Cada nó representa um intervalo $[L,R]$.

::right::

- Divisão: Se um nó representa $[L,R]$, seus filhos representam $[L,mid]$ e $[mid+1,R]$, onde:  
$$mid = L + \lfloor (R-L) / 2 \rfloor$$
- Complexidade:
  - Construção: $\mathcal{O}(n)$
  - Query/Update: $\mathcal{O}(\log n)$
  - Espaço: $\mathcal{O}(n)$

# Seg Lazy

Às vezes, point update não é suficiente. Queremos poder adicionar um valor à todo mundo no intervalo ou setar um valor pra todo mundo no intervalo.

Para isso, podemos usar uma técninca chamada de Lazy Propagation (Propagação Preguiçosa).


::right::

A ideia principal é que não precisamos atualizar todo mundo de uma vez só. Podemos ter updates pendentes em cada nó e, somente quando descermos em uma consulta, propagamos a atualização.

Isso garante que cada update ainda permaneça em $\mathcal{O}(\log n)$
